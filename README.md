# VupSlashWeb
- https://vupslash.icu
- 基于vue3的VUP杀官方网站
- The official website of VupSlash depend on vue3

## Project setup
```
yarn install
```

### Compiles and hot-reloads for development
```
yarn serve
```

### Compiles and minifies for production
```
yarn build
```

### Lints and fixes files
```
yarn lint
```
